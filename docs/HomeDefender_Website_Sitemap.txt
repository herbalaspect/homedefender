How to read this document
This is the complete map of every URL on homedefenderusa.com once SEO work is complete. Legend
Each page in the sitemap below is marked with one of four status tags:
EXISTING
Page is live now and doesn't need substantive changes. Site-wide template fixes (footer, schema, brand name) still apply, but page-specific rewrites aren't required.
EDIT
Page is live but needs the title tag, meta description, H1, and copy rewrites in the Page-by-Page SEO Rewrites document. Full instructions per page are in that document.
NEW
Page does not exist yet and needs to be built. Full webpage copy is in the Website Content Library document — Section 1 for The Defender Shield, Section 3 for city pages, Section 4 for priority pages, and Section 5 for blog posts.
RENAME
Page is live but the URL slug needs to change for SEO consistency. Requires a 301 redirect from the old URL to the new URL so existing links and search rankings transfer.

XML Sitemap & Indexing
In addition to this human-readable sitemap, the site needs a properly configured XML sitemap that Google can crawl. Webflow generates one automatically at /sitemap.xml — verify it's enabled and contains every page below.
After publishing each new page
Verify the page appears in the Webflow-generated /sitemap.xml
Submit the new URL to Google Search Console: URL Inspection tool → Request Indexing
Check the page renders correctly on mobile and desktop
Verify schema markup with Google's Rich Results Test
Add internal links from at least 2 other relevant pages on the site
Recommended internal linking pattern
Strong internal linking is what tells Google the structure of your site. Each page type should link to specific other page types:
Homepage links to: services hub, /defender-shield, top 3–5 city pages, top 3 blog posts
Each city page links to: 3–4 neighboring city pages, the relevant service category page (/service/roofing), and the pricing page
Each roofing sub-service page links to: 2–3 city pages, related sub-service pages (e.g. roof replacement → roof inspection), and 1–2 relevant blog posts
Each blog post links to: 2–3 service pages relevant to its topic, plus 1–2 related blog posts
The Defender Shield page links to: storm damage page, roof inspection page, roof replacement page, and the homepage
Pricing page links to: roof replacement, every relevant city page, financing FAQ
